# Project Aurora :3

Hiyas! Alexi here to explain the project!

Project Aurora is an operating system I'm building to learn how operating systems work and to challenge myself with a fun, long-term programming project. This is a hobby project, so I can't promise continuous or long-term development, but I'll keep working on it whenever I have the time and motivation.

Project Aurora is open source under the GNU GPL v3.0 License. You're welcome to study, modify, and contribute to the project. If you use Aurora's code in your own project, please make sure you follow the terms of the GPL.

The goal of Project Aurora is to create a highly customizable operating system that lets users choose how they want to use their computer. Whether that's a command-line interface, a lightweight window manager, or a full desktop environment, Aurora is designed around flexibility and user choice.

Long-term goals include compatibility with Linux and DOS applications through dedicated compatibility layers. Maybe someday I'll even take on the challenge of adding some level of Windows compatibility!

As the project grows, this README will be expanded with screenshots, build instructions, documentation, and development updates.

Thanks for checking out Project Aurora! ❤️
